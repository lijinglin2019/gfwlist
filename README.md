# gfwlist

在 [gfwlist](https://github.com/gfwlist/gfwlist) 的基础上添加了一些 `gfwlist` 未收录的和大陆访问较慢的地址。

## 使用方法

与 `gfwlist` 的 [使用方法](https://github.com/gfwlist/gfwlist/wiki) 一致。
